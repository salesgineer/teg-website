# Car Pre-Inspection Parameters Research

**Date:** 2025-11-08  
**Purpose:** Research technical parameters for interactive inspection dashboard in hero section  
**Goal:** Identify 6-10 quantifiable parameters with industry standards/norms for interactive display

## Research Overview

This folder contains comprehensive research on technical parameters tested during car pre-purchase inspections. The goal is to create an interactive dashboard that demonstrates TEG's technical expertise while educating users about what should be checked.

## Documents

1. **parameter-specifications.md** - Detailed specifications for each parameter (ranges, norms, thresholds)
2. **dashboard-design.md** - Design and UX considerations for the interactive dashboard
3. **industry-standards.md** - Industry standards and best practices
4. **implementation-plan.md** - Technical implementation plan
